# Como executar os códigos

Para rodar um dos exercícios da lista de typescript, basta seguir os seguintes passos.

### 1. Compilar o projeto
```bash
npx tsc
```

### 2. Executar o exercício desejado
Obs.: Para rodar um exercício diferente do primeiro, basta substituir o número.
```bash
node dist/exercicio1.js
```
